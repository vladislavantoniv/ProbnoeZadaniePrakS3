ЛИС - REST API

Эндпойнты которые можно посмотреть:

Пациенты

GET /api/patients
Получение списка пациентов
GET /api/patients/{id}
Получение пациента по ID (пример с ID=1)

Заявки

GET /api/orders
Получение списка заявок
GET /api/orders/{id}
Получение заявки по ID (пример с ID=1)
GET /api/orders/patient/{patientId}
Получение всех заявок конкретного пациента (пример с patientId=1)

Типы анализов

GET /api/test-types
Получение списка типов исследований
GET /api/test-types/{id}
Получение типа исследования по ID (пример с ID=1)

Тесты

GET /api/tests
Получение списка тестов
GET /api/tests/{id}
Получение теста по ID (пример с ID=1)
GET /api/tests/order/{orderId}
Получение всех тестов конкретной заявки (пример с orderId=1)
